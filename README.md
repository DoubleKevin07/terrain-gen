# Terrain Generation

This is a library for generating realistic looking terrain heightmaps (and possibly more in the future).

## Installation

1. Install JDK
2. Install Gradle
3. Clone this repository
4. Run `gradle build`
5. Run `gradle run`

(Optional) Run `gradle test` to execute unit tests.
